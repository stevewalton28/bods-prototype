BODS Data Submission Service

Simplified XML submissions for bus operators

Easy capture, validation, and generation of TransXChange XML files

Use this service to:

  • enter your bus service data through a simple form interface
  • validate your data against the official TransXChange schema
  • generate compliant XML files for submission to BODS
  • save time and reduce errors in your submissions

The service guides you through each step of the process, ensuring all required information is captured correctly.

Start now

Before you start

You'll need:

  • your operator details, including licence number
  • service details, including service code and line number
  • route details and stop points information
  • journey timings and frequency information

This demonstration service does not actually submit data to BODS, but generates valid XML files that you can download and submit manually.
